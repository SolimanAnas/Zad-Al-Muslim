# 🌙 Zad Al-Muslim | زاد المسلم

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen.svg?style=for-the-badge)](https://solimananas.github.io/Tasbee7)
[![GitHub license](https://img.shields.io/github/license/SolimanAnas/Tasbee7.svg?style=for-the-badge)](https://github.com/SolimanAnas/Tasbee7/blob/main/LICENSE)

**Zad Al-Muslim** is a modern, lightweight, and ad-free Progressive Web App (PWA) designed to be the perfect spiritual companion for Muslims. It features a beautiful glassmorphism UI, smooth animations, and offline capabilities.

**زاد المسلم** هو تطبيق ويب تقدمي (PWA) عصري، خفيف، وخالٍ تماماً من الإعلانات، تم تصميمه ليكون الرفيق الروحي المثالي. يتميز بواجهة زجاجية أنيقة (Glassmorphism)، انتقال سلس بين الشاشات، ويدعم العمل بدون إنترنت.

---

## ✨ Features | المميزات

* **📖 Holy Quran (تصفح المصحف):** Read the Quran with a clean, distraction-free interface.
* **🎧 Quran Audio Player (الاستماع للقرآن):** Listen to multiple reciters with a smart player that supports background audio, media sessions, and favorites.
* **📿 Smart Masbaha (المسبحة الذكية):** Digital counter with haptic feedback, customizable daily targets, lifetime statistics, and a unique "Power Saving Mode" for OLED screens.
* **📻 Live Quran Radio (إذاعة القرآن):** 24/7 live broadcasts from various Quran radio stations.
* **🤲 Azkar & Hisn Al-Muslim (الأذكار وحصن المسلم):** Morning, evening, and Ruqyah supplications with interactive counters.
* **🎨 Modern UI (واجهة عصرية):** 3-Mode Theme System (Dark Glass, Light Glass, Plain Dark) that remembers your preference.
* **📱 PWA Ready (تطبيق متكامل):** Installable on iOS and Android directly from the browser for a native app-like experience.

## 🛠️ Tech Stack | التقنيات المستخدمة
* **Frontend:** HTML5, Vanilla JavaScript, CSS3 (Custom Glassmorphism Design).
* **Audio Streaming:** HLS.js for live radio and direct MP3 streaming.
* **Storage:** Native browser `localStorage` for saving settings, bookmarks, and statistics.
* **PWA:** Service Workers and Web App Manifest for offline caching and installation.
